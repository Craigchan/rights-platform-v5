## 🚀 快速开始

### 安装依赖
```bash
cd rights-platform-v3
pnpm install
```

### 开发模式
```bash
pnpm run dev
```
访问 http://localhost:5173

### 构建生产版本
```bash
pnpm run build
```
输出目录：`dist/`

### 预览生产版本
```bash
pnpm run preview
```
