SVG路径解析:

M 0,0 - 从左上角开始

L 1,0 - 直线到右上角

L 1,0.65 - 直线向下到65%高度

A 0.5,0.35 0 0,1 0,0.65 - 椭圆弧到左侧65%高度

Z - 闭合路径

参数说明:

rx=0.5 (水平半径50%)
