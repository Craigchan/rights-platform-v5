三次贝塞尔曲线(C命令):

C 0.75,1 0.25,1 0,0.5

• 控制点1: (0.75, 1) - 右下方

• 控制点2: (0.25, 1) - 左下方

• 终点: (0, 0.5) - 左侧50%高度

• 弧度深度: 50% (从50%到100%)
