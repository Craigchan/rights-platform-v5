三种轮播样式说明
样式一：沉浸式文字叠加

适用场景：政府补贴、重大活动、品牌宣传
特点：大标题 + 渐变背景，视觉冲击力强，信息传达直接

样式二：卡片式图文混排

适用场景：优惠券领取、积分活动、会员权益
特点：左侧图标 + 右侧文字，信息层次清晰，可展示多维度信息

样式三：极简卡片式

适用场景：积分商城、附近商家、推荐内容
特点：大图展示，极简设计，类似电商卡片
